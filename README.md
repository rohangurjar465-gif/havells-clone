# Havells Clone Website

## 📌 About Project
This is a clone of the Havells official website created using HTML, CSS, Bootstrap, and JavaScript.

## 🚀 Features
- Responsive Navbar
- Image Carousel/Slider
- Product Sections
- Responsive Design
- Modern UI
- Bootstrap Components
- Mobile Friendly Layout

## 🛠 Technologies Used

- CSS3
- Bootstrap 5
- JavaScript
